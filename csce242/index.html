Assignments

Assignment 1 - Basic HTML

This assignment involved creating a basic HTML page featuring images, a table, and essential HTML elements.

Assignment 2 - Basic CSS

This assignment focused on CSS, including styling navigation, images, and tables.

Assignment 3 - Page Layout

This assignment focused on using Flexbox to create a flexible webpage layout.

Assignment 4 - Recreate CSS Page (USC)

This assignment involved recreating the USC Graduate School Admissions page, focusing on replicating the layout and structure using HTML and CSS.

Assignment 5 - JavaScript Buttons, Functions, and More JavaScript Buttons, Functions, and More

This assignment involved creating a responsive webpage with a three-column layout, where clicking a column increments a counter, refreshing an image displays a new one, and adjusting a slider moves a square proportionally across the page.

Assignment 6 - Javascript Conditionals

This assignment involved implementing JavaScript conditionals to change the content of the webpage based on user interactions with a color slider and picture chooser, ensuring it is responsive.

Assignment 7 - Loops

For the stars assignment, I implemented JavaScript to draw stars on the webpage based on user input. The stars appear in random positions within a square, and clicking a star reveals a message indicating which star was selected.

Assignment 8 - Arrays

In this assignment, I implemented a webpage where clicking on images displays a title and description using JavaScript and an associative array.

Assignment 9 - Classes

In this assignment, I built a webpage displaying bird information using JavaScript classes. Each bird has its own class with details like name, size, and lifespan. When a bird image is clicked, a modal dialog appears showing all the bird's information.

Assignment 100 - Parsing JSON

In this assignment, I parsed a JSON file to display ice cream images and their names. When an image is hovered over, the name of the ice cream appears. The images and data were retrieved from a provided link and styled according to the requirements.

Assignment 11 - Hello World (Server Side)

For this assignment, I made a Node.js server with a "Hello World" message, deployed it on Render, and added clickable links to see the code on GitHub or live code Render.

Project Part 4/5: Colors, Pictures, and Text / Updated Feedback

I accomplished this by adding relevant images, updating the text, using media queries for responsiveness, and fixing the navigation for a more user-friendly experience.

Project Part 6: JSON File Parsing

For Project Part 6, the goal was to create a dynamic webpage that loads and displays information using a JSON file. I accomplished this by first creating a JSON file containing a list of fitness classes offered, with each item containing a unique ID, a relative link to an image, an array of additional information, and at least four other fields such as confirmation number, instructor level, availability, and location.Along with addressing comment suggestions made by my professor and TAs, I made necessary improvements.

Project Part 7: Client Side Form validation and JavaScript

For Project Part 7, I created a functional Contact Me form with validation that displays success or error messages without redirecting. I also embedded a responsive Google Map for the University of South Carolina, ensuring both the form and map look good on all screen sizes.
